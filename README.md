# Genius Scan File Downloader

> Download files from the mobile application Genius Scan

# Install

```
$> npm install -g gs-download
```

# Usage

* `--host | -h`: The host
* `--port | -p`: The port
* `--target | -t`: The target where files are downloaded

```
$> gs-download -h 192.168.199.102 -p 8080 -t ~/Genius
```
